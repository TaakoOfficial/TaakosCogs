# Changelog

## 1.1.1

- Renamed the repository package folder from `Giveaway` to `giveaway` to match Red cog naming standards.
- New installs should use `[p]cog install taakoscogs giveaway` and `[p]load giveaway`.
- Existing giveaway settings and tracked giveaway records are preserved because the cog's Config identifier did not change.

## 1.1.0

- Added Red-Web-Dashboard third-party integration for Giveaway.
- Added dashboard controls for starting giveaways, attaching existing messages, ending, cancelling, rerolling, refreshing status messages, and viewing tracked giveaway records.

## 1.0.0

- Initial release
- Added timed reaction-based giveaways
- Added automatic ending, cancellation, rerolls, and active giveaway listing
