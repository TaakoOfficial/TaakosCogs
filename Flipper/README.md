# Flipper

Flipper is a Red DiscordBot cog providing a simple coin flip command.

## Features

- `coinflip` command: Flip a coin and get Heads or Tails.

## Installation

1. Place the `Flipper` (formerly Dicey) folder in your `cogs` directory.
2. Load the cog with:
   ```
   [p]load flipper
   ```
