# Changelog

## 1.2.0

- Added Red-Web-Dashboard integration for TempVoice settings, active channel review, control panel resend, and empty-channel cleanup.

## 1.1.0

- Added a native Discord member picker to transfer, permit, and remove modals.

- Initial release.
- Added join-to-create temporary voice channels.
- Added embedded control panels with rename, lock/unlock, user limit, transfer, permit, remove, and claim controls.
- Added automatic cleanup for empty temporary voice channels.
