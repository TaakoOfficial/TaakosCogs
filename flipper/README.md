# flipper

A small Red-DiscordBot utility cog for flipping a coin.

[Back to the cog catalog](../README.md)

## Install

```text
[p]repo add taakoscogs https://github.com/TaakoOfficial/TaakosCogs
[p]cog install taakoscogs flipper
[p]load flipper
```

## Dashboard

This cog includes standalone Red-Web-Dashboard integration. Server managers can use the dashboard to view visible commands and the current server configuration when Red-Web-Dashboard is installed.

## Highlights

- Simple `Heads` or `Tails` coin flip.
- Color-coded embed result.
- Hybrid command support for prefix and slash usage.
- No external dependencies.

## Commands

| Command                      | Description                                     |
| ---------------------------- | ----------------------------------------------- |
| `[p]coinflip` or `/coinflip` | Flip a coin and display the result in an embed. |

## Requirements

- Red-DiscordBot 3.0.0 or newer.
- `Send Messages` and `Embed Links` permissions.

## Data

Flipper does not persistently store end user data.
