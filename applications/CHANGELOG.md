# Changelog

## 1.3.1

- Renamed the repository package folder from `Applications` to `applications` to match Red cog naming standards.
- New installs should use `[p]cog install taakoscogs applications` and `[p]load applications`.
- Existing forms, panels, responses, polls, and setup data are preserved because the cog's Config identifier did not change.

## 1.3.0

- Added Red-Web-Dashboard third-party integration for Applications.
- Added dashboard controls for application creation, deletion, settings, role lists, questions, panels, responses, and polls.
- Added dashboard response review actions for accepting and denying pending applications.
